### Ohjeet
#### Vaatimukset:
- Python versio 3.10 tai uudempi
- Poetry

#### Asennus:
- Kloonaa tai lataa repositorio
- Avaa projektin juurihakemisto terminaalissa
- Suorita komento `poetry install`

#### Käyttö:
- Avaa projektin juurihakemisto terminaalissa
- Suorita komento `poetry run python src/main.py`
